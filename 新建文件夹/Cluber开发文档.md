# 开发文档

### **开发准备阶段（第 1-3 天）**

| 日期    | 工作内容                                                     | 产出物                                            |
| ------- | ------------------------------------------------------------ | ------------------------------------------------- |
| 第 1 天 | 1. 搭建项目基础框架（Vue3 + Vite）2. 配置路由（router/index.js）、Pinia（stores/index.js）3. 安装依赖（axios、element-plus 等）4. 创建目录结构（按文档中的 src / 子目录） | 可运行的空项目框架，完整目录结构，基础配置文件    |
| 第 2 天 | 1. 封装通用工具函数（utils/request.js：axios 拦截器、token 处理）2. 实现全局样式（assets/css/global.css）3. 注册全局组件（common/Button、Input、Dialog） | 工具函数文档，全局样式文件，3 个基础通用组件      |

### **说明**

1. 每日工作时间按 8 小时计算，包含编码、调试、自测环节。
2. 若遇复杂 bug 或需求调整，可灵活调整相邻模块的开发时间。
3. 优先保证核心功能（登录、社团管理、活动管理）的开发质量，次要功能（如推送管理）可适当压缩时间。
4. 测试阶段需重点关注权限控制、数据一致性（如申请状态更新后相关列表同步）等关键逻辑。
