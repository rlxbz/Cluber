# Cluber

一个基于 Vue 3 的社团活动平台业务前台，面向学生和社团管理员，聚焦社团浏览、活动报名、申请提交、内容查看与个人服务等真实前台业务流程。

## 项目定位

- 面向学生用户，提供社团和活动的浏览、报名、申请与个人中心能力
- 面向社团管理员，补充本社团范围内的轻业务处理能力
- 重点体现前台体验、业务流转、状态反馈与角色边界，而不是平台级总控能力

## 项目简介

当前仓库覆盖的主线业务包括：

- 社团浏览与社团详情查看
- 活动浏览、活动详情查看与活动报名
- 入社申请、建社申请、活动申请的提交与进度查看
- 公告与社团动态浏览
- 个人中心、我的社团、我的申请、界面偏好

其中 `club_admin` 在项目中只承担“本社团相关轻业务”的一部分，例如入社申请回复、我的社团服务、活动申请发起等；更强的平台管理能力不再作为本项目主线。

## 角色说明

| 角色 | 当前定位 | 主要能力 |
| --- | --- | --- |
| `student` | 主角色 | 浏览社团、查看活动、报名活动、提交申请、查看公告与动态、维护个人资料、查看我的申请 |
| `club_admin` | 扩展角色 | 保留学生端能力，并处理本社团范围内的轻业务，如入社申请回复、我的社团服务、活动申请发起 |
| `sys_admin` | 弱化体验角色 | 当前仅保留界面体验与调试入口，不是 Cluber的核心角色，也不是本仓库的主要展示重点 |

## 功能模块

### 1. 首页与内容浏览

- 首页聚合热门活动、公告与社团动态
- 统一提供前台化的 loading / empty / error 状态
- 支持游客查看基础内容

### 2. 社团广场

- 浏览社团列表与社团详情
- 查看社团介绍、成员服务信息与活动概览
- 登录后可发起入社申请，并获得即时反馈

### 3. 活动中心

- 浏览活动列表与活动详情
- 查看报名状态、活动时间与活动说明
- 登录用户可报名活动，成功或失败都有明确反馈

### 4. 申请服务

- 提交入社申请、建社申请、活动申请
- 查看我的申请记录与处理状态
- 社团管理员可在前台完成“本社团相关”的轻量申请回复

### 5. 公告与动态

- 查看公告列表、公告详情
- 查看社团动态列表与动态详情
- 在详情页内补齐无数据、异常、无权限等状态反馈

### 6. 我的页面

- 维护个人资料
- 查看我的社团
- 查看我的申请
- 进入与自己相关的社团服务页

### 7. 界面偏好

- 支持主题、字体等界面偏好设置
- 作为前台体验的一部分保留，不承载后台系统配置职责

## 技术栈

- `Vue 3`：前端框架
- `Vite`：开发与构建工具
- `Vue Router`：前端路由与页面权限接入
- `Pinia`：状态管理
- `Element Plus`：基础 UI 组件
- `Axios`：请求封装
- `dayjs`、`date-fns`：日期处理
- `@element-plus/icons-vue`：图标资源
